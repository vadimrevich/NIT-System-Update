# ПАРТНЕРСКАЯ ПРОГРАММА

## КАК СТАТЬ ПАРТНЁРОМ ПРОЕКТА

- Зарегистрируйте хостинг в Интернете с круглосуточной доступностью и доменным именем.
- Создайте в его корне папку Exponenta, в которой будут располагаться основные файлы проекта.
- Свяжитесь с автором для получения оригинальных ссылок на дистрибутивы, или скачайте их полные версии с сайтов производителей программного обеспечения.
- Создайте страницу с ссылками на файлы дистрибутивов, их контрольных сумм и лицензиями на программное обеспечение.
- Отправьте файл со ссылками на дистрибутивы автору проекта на регистрацию. Без этой регистрации использование данного зеркала не является законным.
- Можете использовать данный сайт для распространения программного обеспечения, а также для кастомизации и написания новых модулей к файлам проекта.
- При создании зеркала проекта ссылка на авторство продукта и файл с лицензией обязателен

## ЗАМЕЧАНИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ

Данное программное обеспечение может опознаваться антивирусным программным обеспечением как Hack-Tools.not-a-virus (как хакерское программное средство для удалённого администрирования). Это нормальное обозначение для такого рода программного обеспечения для административных нужд. Пожалуйста, не помечайте вручную это программное обеспечение как вредоносное, чтобы исключить ложное срабатывание антивируса.

На текущий момент антивирус Avast производит ложное срабатывание на файлах данного проекта, указывая на внедрённый в инсталлятор вредоносный код Malware или Trojan. В связи с этим автор крайне не рекомендует совместное использование этих административных утилит и антивируса Avast Free Antivirus. В случаен, если Вы хотите использовать данное программное обеспечение совместно, добавьте исключения антивируса на инсталлируемые каталоги с программами проекта (небезопасно).

Для совместного использования антивируса и файлов проекта автор рекомендует стандартные антивирусы от Microsoft: Microsoft Security Essential и Windows Defender.
